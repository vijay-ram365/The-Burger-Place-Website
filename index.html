About The Burger Place

The Burger Place is actually a mock up for a restaurant/fast-food business website to demonstrate HTML and CSS principles. This website features basic HTML structure, links, maps, icons, and images while the CSS demonstrates basic design and responsiveness with Flexbox and Media Queries.

Contact Us!

The Contact section features links to popular social media sites and is designed to be adjusted based on how much icons and links need to be placed. This is a great section for businesses to also mention their own social media activity and posts.
